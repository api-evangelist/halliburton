# Halliburton (halliburton)

Halliburton delivers advanced products and services to meet the evolving needs of the global energy industry, driving innovation, reliability, and performance. Halliburton operates enterprise software platforms such as DecisionSpace 365, Landmark, and iEnergy, but does not currently publish a public, openly accessible REST API for general developer use.
